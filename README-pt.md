# DsCommerce

* Read in [English](https://github.com/kelvin-feltrin/dscommerce/blob/main/README.md)

## Sobre o projeto
O "DsCommerce" é um projeto de venda online de produtos. Ele aplica os seguintes conceitos:
- Criação de componentes;
- Navegação por rotas;
- Sistema de carrinho de compras armazenado localmente;
- Lista de produtos com paginação e filtros;
- Validação de campos e exibição de erros usando a técnica de campos "sujos";
  
Projeto baseado no curso de Nelio Alves: [ReactJS Professional](https://devsuperior.com.br/curso-reactjs-professional)
